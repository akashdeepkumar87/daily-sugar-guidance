# Daily Sugar Guidance App

A simple and user-friendly web application that provides daily diet and walking
guidance based on a user's morning blood sugar level.

## Features
- Easy sugar input
- Clear visual guidance using icons and colors
- Suitable for both literate and low-literacy users
- Works on desktop and mobile browsers

## Tech Stack
- Python
- Streamlit

## How It Works
1. User enters morning blood sugar level
2. App analyzes the value using safe rules
3. User receives diet and activity guidance for the day

## Disclaimer
This app provides general guidance only and does not replace medical advice.
